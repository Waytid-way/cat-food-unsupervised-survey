สำรวจบรรจุภัณฑ์อาหารแมว

ระบบแคมเปญการตลาดด้วย AI สำหรับ EDA, การแบ่งกลุ่ม, การพยากรณ์ และการตัดสินใจทางธุรกิจ

แปลงคำตอบแบบสำรวจเป็นกลุ่มลูกค้า → จากนั้นแปลงกลุ่มเป็นระบบให้คะแนนสำหรับการตลาด

คำถามทางธุรกิจ

  • การออกแบบบรรจุภัณฑ์ใดเหมาะกับกลุ่มลูกค้าใด?
  • เราสามารถเปลี่ยนข้อมูลเชิงลึกจากแบบสำรวจเป็นระบบที่ให้คะแนนลูกค้าใหม่โดยอัตโนมัติได้หรือไม่?

สัญญาของระบบ

  • EDA บอกเราว่าข้อมูลมีลักษณะอย่างไร
  • Unsupervised learning ค้นหากลุ่มลูกค้าและความผิดปกติ
  • Supervised learning เปลี่ยนกลุ่มเหล่านั้นเป็นโมเดลพยากรณ์
  • ข้อมูลเชิงธุรกิจแปลงผลลัพธ์ของโมเดลเป็นการดำเนินการแคมเปญ

End-to-end flow

Raw survey -> EDA -> Unsupervised segmentation -> Supervised classifier -> Dashboard + business action

ภาพรวมข้อมูล

  • แหล่งที่มา: ส่งออกจาก Google Form แบบสำรวจบรรจุภัณฑ์อาหารแมว
  • แถวดิบที่โหลด: 167 แถว
  • คำตอบ top-3 ที่สมบูรณ์ที่ใช้สำหรับการสร้างโมเดล: 148 คำตอบ

สิ่งที่อยู่ในแบบสำรวจ

  • ข้อมูลประชากร
  • ปัจจัยการซื้อ
  • ความสำคัญของบรรจุภัณฑ์
  • การให้คะแนนตัวเลือก 10 รูปแบบบรรจุภัณฑ์

  • 50 Likert rating columns

  • 5 demographic columns

Unsupervised Overview

ทำไมต้องเริ่มจาก unsupervised

  • เราไม่รู้กลุ่มลูกค้าที่แท้จริงก่อนการสร้างโมเดล
  • แบบสำรวจออกแบบมาเพื่อค้นหารูปแบบ ไม่ใช่แค่พยากรณ์ label

Pipeline

Encode -> Ipsatize -> Impute -> PCA -> K-Means -> Hierarchical validation -> Isolation Forest

สิ่งที่ได้จากขั้นตอนนี้

  • customer segments
  • anomaly flags
  • a business-friendly persona view

Reference outputs

  • outputs/metrics_summary.json
  • outputs/segment_profiles.csv
  • reports/unsupervised_report_th.md

Unsupervised Feature Engineering

สิ่งที่โค้ดทำ

  • แปลง top-3 rankings เป็น weighted vote features
  • ลด response-style bias ด้วย ipsatization
  • เติม missing buy-factor values ก่อน clustering
  • สร้าง merged feature matrix สำหรับ clustering และ anomaly detection
