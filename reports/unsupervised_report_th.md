# รายงานการเรียนรู้แบบไม่มีผู้สอนเพื่อแบ่งกลุ่มผู้บริโภค

## ภาพรวมวิธีการ
- หลังคัดกรองได้ผู้ตอบครบถ้วน 148 คน
- ใช้ feature แบบ ipsatized จาก option-rating 50 ตัวแปร แล้วลดมิติด้วย PCA 8 components
- final_cluster_key = `segment` และ final_cluster_k = 2
- เลือกคลัสเตอร์สุดท้ายที่ `k = 2` โดยใช้ `segment` เป็น label

## Customer Persona และข้อเสนอเชิงกลยุทธ์
### Segment 1 - กลุ่มพรีเมียมเน้นคุณภาพและดีไซน์
- ขนาดกลุ่ม: 36 คน (24.3%)
- เพศเด่น: หญิง
- อายุเด่น: 30-39ปี
- สถานภาพเด่น: โสด ไม่มีแฟน
- ตัวเลือกที่ชอบมากสุด: Option 03 (4.31), Option 04 (3.96), Option 02 (3.64)
- ปัจจัยตัดสินใจเด่น: รสชาติกลมกล่อมอร่อยถูกปากแมว 4.61, แบรนด์มีชื่อเสียงเป็นที่รู้จัก 4.50, ใช้วัตถุดิบจากธรรมชาติ 4.31
- อัตรา anomaly: 19.4%
- กลยุทธ์: สื่อสารแบบ premium ให้ชัดเจน เน้นรสชาติ วัตถุดิบธรรมชาติ และสัญลักษณ์ที่บอกประโยชน์ของสินค้า

### Segment 2 - กลุ่มตลาดหลักที่ชั่งน้ำหนักหลายปัจจัย
- ขนาดกลุ่ม: 112 คน (75.7%)
- เพศเด่น: หญิง
- อายุเด่น: 30-39ปี
- สถานภาพเด่น: แต่งงานแล้ว
- ตัวเลือกที่ชอบมากสุด: Option 03 (3.62), Option 06 (3.43), Option 07 (3.39)
- ปัจจัยตัดสินใจเด่น: รสชาติกลมกล่อมอร่อยถูกปากแมว 4.38, ใช้วัตถุดิบจากธรรมชาติ 4.25, แบรนด์มีชื่อเสียงเป็นที่รู้จัก 3.92
- อัตรา anomaly: 17.0%
- กลยุทธ์: สื่อสารแบบ premium ให้ชัดเจน เน้นรสชาติ วัตถุดิบธรรมชาติ และสัญลักษณ์ที่บอกประโยชน์ของสินค้า

## สรุปสำหรับการนำเสนอ
- Segment 1 เป็นกลุ่มที่ให้คะแนนเรื่องรสชาติ วัตถุดิบธรรมชาติ และสัญลักษณ์บนแพ็กเกจสูงกว่า จึงเหมาะกับ messaging แบบ premium และชัดเจน
- Segment 2 คือฐานผู้บริโภคหลัก ควรสื่อสารแบบกว้าง เน้นความคุ้มค่า ความน่าเชื่อถือ และข้อมูลสินค้าที่อ่านง่าย
- โมเดลโดยรวมให้ภาพที่ใช้งานได้ แต่ยังมี overlap จึงควรใช้ร่วมกับ insight เชิงธุรกิจ ไม่ควรตีความว่าเป็นกลุ่มแยกขาดกัน 100%
